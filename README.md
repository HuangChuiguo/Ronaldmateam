# DKD target discovery and drug repurposing

Pre-release placeholder repository for analysis code supporting the manuscript
"A multi-omics and real-world evidence framework for target discovery and drug
repurposing in diabetic kidney disease."

## Status

The analysis code is undergoing internal review and is not yet available. This
repository currently contains only the planned directory structure. It will be
populated before publication.

No individual-level clinical data or other restricted data will be deposited in
this repository.

## Planned structure

- `01_target_nomination/`: multi-omics gene scoring and network-based nomination
- `02_scorecard_prioritization/`: candidate scorecard and weighting analyses
- `03_genetic_liability_assessment/`: phenome-wide and ageing-related analyses
- `04_approved_drug_mapping/`: target-to-approved-drug mapping
- `05_clinical_outcome_evaluation/`: routine-care clinical outcome analyses
- `06_figure_generation/`: scripts used to generate manuscript figures
